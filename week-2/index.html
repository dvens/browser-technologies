CSS - Feature: Flexbox (Internet Explorer Fallback)

Flexbox is a new layout mode in CSS3. Flexbox provides an improvement over the block model in that it does not use floats. The fallback are html comments above the html tag.

Can I Use?

  • Can I Use Flexbox?
  • Flexbox Cheatsheet

Code & Fallback

If display flex is not supported .flex class will fallback to display: block;. .flex { display: block; display: flex; flex-direction: row; flex-wrap: wrap; justify-content: space-between; }

.flex-item { height: 100px; background-color: white; border: 2px solid cadetBlue; flex-basis: 48% }

Above your html tag use the following code that recognizes IE8 and IE9 without the single(') quotes <'!--[if IE 8]'> <'html class="no-js lt-ie10 lt-ie9"> <'![endif]--'>
<'!--[if IE 9]'><'html class="no-js lt-ie10"><'![endif]--'>

It will add the class lt-ie10 to the html tag so you can use the following code as fallback. Other tip is to use modernizr(it recognizes more browsers) as fallback, so you could use .no-flexbox as selector. // Fallback for ie8 & ie9
.lt-ie10 .flex-item { float: left; margin-right: 2%; width: 48% }

.lt-ie10 .flex-item.last { margin-right: 0; }

CSS - Feature: Calc()

The calc() function is used to calculate and set the outcome as value in a css-property.

Can I Use?

  • Can I Use Calc()
  • Use Cases for Calc()

Code & Fallback

If the browser does not support calc it will fallback on the width that is above the width property that uses calc().

.calc { height: 20px; background-color: white; border:2px solid cadetBlue; width: 500px; width: calc(50% - 20px); }
