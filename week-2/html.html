HTML - Feature: Picture element

The picture element contains multiple source elements that will determine on which browser size an specific image will be shown. If the picture element is not supported, the fallback will be the defined img tag.

Can I Use?

  • Can I Use picture element?

Code & Fallback

<picture> <source media="(max-width: 580px)" srcset="img/header-580.jpg"> <source media="(max-width: 880px)" srcset="img/header-880.jpg"> <source media="(max-width: 1000px)" srcset="img/header.jpg"> <img src="img/header.jpg" alt="House of cards"> </picture>

HTML - Feature: input[type=data]

The geolocation API allows the user to provide their location to web applications. It will ask the user to allow their browsers to access the location with a popup box.

Can I Use?

  • Can I Use input type date?

Code & Fallback

If the input type is not supported it will fallback to type text and it will show the placeholder value as fallback. <input type="date" placeholder="dd-mm-yyyy">
